# Piezo Sonar Lab MVP

## 何をしたか

- 既存の耐圧計算アプリのエントリを、ピエゾ素子の特性同定とソナー適性診断を行う1ページアプリへ差し替えた。
- 左ステップナビ、中央の解析ビュー、右インサイトパネルで、測定から次の実験提案までの流れを実装した。
- Rechartsによるインピーダンス/位相グラフ、共振ピーク表、BVD等価回路模式図、SVG振動モード可視化、ソナー適性スコアを追加した。
- CSV簡易パースとサンプルデータ読み込みを実装した。

## 何を決めたか

- 既存の未使用耐圧計算コードは削除せず、トップページの表示対象だけを新アプリへ変更した。
- 既存`src/lib/types.ts`は旧計算機が使っているため、ピエゾ用型は同ファイルへ追加エクスポートした。
- RechartsのSSR警告を避けるため、グラフ描画はクライアントマウント後に行う。

## ハマりどころと解決

- `node_modules`のpnpmリンクが壊れており、`pnpm lint`がeslint実体なしで失敗した。`npm install --package-lock=false`で依存解決を復旧した。
- 旧`PressureCaseApp`がReact 19の新しいlintルールに引っかかったため、挙動を変えずに該当ルールをファイル単位で無効化した。
- RechartsがSSR時にコンテナサイズ警告を出したため、マウント後描画に変更した。
